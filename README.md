# Healthcare Cost Analysis

This project explores healthcare expenditure trends, identifies key cost drivers, and develops strategies for cost containment using a dataset from Kaggle. The analysis is conducted using PostgreSQL for database management, Python for data analysis, and Tableau for data visualization.
